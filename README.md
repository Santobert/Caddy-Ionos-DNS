# Caddy-Ionos-DNS

Inofficial Caddy Container with built-in IONOS DNS plugin and CrowdSec Bouncer

## Included Modules

- **IONOS DNS**: https://github.com/caddy-dns/ionos
- **CrowdSec Bouncer (HTTP)**: https://github.com/hslatman/caddy-crowdsec-bouncer/http
- **CrowdSec Bouncer (Layer4)**: https://github.com/hslatman/caddy-crowdsec-bouncer/layer4
- **CrowdSec Bouncer (AppSec)**: https://github.com/hslatman/caddy-crowdsec-bouncer/appsec
