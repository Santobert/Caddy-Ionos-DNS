# Caddy-Ionos-DNS

Inofficial Caddy Container with built-in IONOS DNS plugin and CrowdSec Bouncer

## Included Modules

- **IONOS DNS**: https://github.com/caddy-dns/ionos
- **CrowdSec Bouncer (HTTP, Layer4, AppSec)**: https://github.com/hslatman/caddy-crowdsec-bouncer
